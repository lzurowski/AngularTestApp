#####################################
#### APLIKACJA REJESTR PRODUKTÓW ####
#####################################

Stworzyć aplikacje umożliwiająca zarządzającą tabelką produkty.

Aplikacja ma umożliwić wykonywanie podstawowych operacji CRUD na rekordach tabeli.

W głownym widoku aplikacji powinna znajdować się tabela (GRID) z listą dostepnych produktów.
Nastepnie przy kazdym produkcjie musi byc dostepna acja z możliwościa edycji i usunięcia produktu.
Dodatkowo musi byc udostepniony formularz z możliwościa dodania produktu.
Aplikacja powinna działac w opraciu wzorcva: Single-page application

* W celu realizacji zadania należy wykożystac nastepujące technologie:

Warstwa Klienta:
- HTML
- JavaScrip
- Framework AngularJS (obowiązkowy)
- Dowolne innne nażędzia JS, CSS

Warstwa Serwera:
- PHP 
- Dowolny Framework PHP (nieobowiazkowy, preferowany Yii2) 

DB
- Należy wykorzystać przesłana bazę danych SQLite 
- Lub obsługiwać w plik JSON  

* Mile widziane w aplikacj:
- Paginacja w podgladzie tabeli
- Walidacja formularza edycji/dodania rekordu
- Użycie frameworka bootstrap css
- Skożystanie z menadzera Bower "A package manager for the web"
- Skożystanie z composher.php
